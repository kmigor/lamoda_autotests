## <a name="Что проверяют тесты?">**Что проверяют тесты?**</a>

Представленные смок тесты проверяют:
1) Главную страницу
2) Страницу мужских товаров
3) Страницу женских товаров
4) Страницу детских товаров
5) Корзину
